The CUMT-HMDV2 dataset contains five biometric modalities collected from 532 individuals, including finger veins, finger knuckle prints, dorsal hand veins, palmprints, and fingerprints. For each modality, 20 images were captured, resulting in a total of 53,200 high-quality images. The dataset encompasses biometric variations across different genders, ages, and individuals, ensuring sufficient scale and diversity.
