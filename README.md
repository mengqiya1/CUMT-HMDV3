The CUMT-HMDV3 dataset contains five biometric modalities collected from 532 individuals, including finger veins, finger knuckle prints, dorsal hand veins, and palmprints. For each modality, 20 images were captured, resulting in a total of 53,200 high-quality images. The dataset encompasses biometric variations across different genders, ages, and individuals, ensuring sufficient scale and diversity.
