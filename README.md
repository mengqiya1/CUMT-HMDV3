The CUMT-HMDV3 dataset contains four biometric modalities collected from 532 individuals, including finger veins, finger knuckle prints, dorsal hand veins, and palmprints. For each modality, 20 images were captured under diverse illumination conditions, resulting in a total of 42,560 images. The dataset encompasses biometric variations across different genders, ages, and individuals, ensuring sufficient scale and diversity.
